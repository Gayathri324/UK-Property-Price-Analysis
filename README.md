# England & Wales Property Price Analysis

## Overview
This project analyzes property price trends across England and Wales using transaction-level data. The goal is to understand regional differences, property types, and market activity over time.

An interactive dashboard was built using Power BI to present key insights.

---

## Objectives
- Analyze property price trends over time  
- Compare prices across regions  
- Explore property types and their price distribution  
- Examine transaction patterns by tenure  
- Visualize geographical price variations  

---

## Dataset
The dataset includes:
- Date of Transfer  
- Property Price  
- Property Type  
- Region / County  
- Tenure (Freehold / Leasehold)

## Key Insights
- London has the highest property prices compared to other regions  
- Existing properties are generally more expensive than new builds  
- Detached houses have the highest average prices  
- Freehold transactions dominate the market  
- Property prices show long-term growth trends  

---

## Dashboard Features
- Average property price by region  
- Price comparison by property type  
- Map of most and least expensive counties  
- Yearly price trend analysis  
- Quarterly transaction trends by tenure
